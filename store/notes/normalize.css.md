### usage

#### npm

`npm install --save normalize.css`

`import 'normalize.css'`

#### cdn

> See https://yarnpkg.com/en/package/normalize.css  

```html
<script src="https://unpkg.com/normalize.css@8.0.1/normalize.css"></script>
```
