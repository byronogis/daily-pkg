### usage

> 记得向目标区域添加 class, `markdown-body`  

#### npm

```bash
npm install github-markdown-css
```

```js
import 'github-markdown-css'
```

#### cdn

```html
<link rel="stylesheet" href="https://unpkg.com/github-markdown-css@5.1.0/github-markdown.css"/>
```
